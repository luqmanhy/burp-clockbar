## ClockBar
A simple Burp Suite extension that shows a clock at the bottom of the interface.

## Installation
1. Download the [JAR extension file](https://github.com/luqmanhy/burp-clockbar/releases).
2. Open Burp Suite.
3. Navigate to **Extender â†’ Extensions**.
4. Click **Add**.
5. Set **Extension Type** to **Java**.
6. Select the downloaded file: `clockbar.jar`.
7. Click **Next** until the installation is complete.
